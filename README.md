# Streeto — live demo

Pixel art RPG street racing set in Western Sydney. Self-contained static HTML preview.

This is the public deploy of the static preview only. The main project lives in a
private repo. Source-of-truth content is generated by PixelLab and hand-written
JS; this copy is updated as snapshots for phone testing.

## Open it

Open `index.html` in any modern browser. On mobile, use the GitHub Pages URL
attached to this repo.
